This sample shows how to use the Windows.Devices.Sensors.Accelerometer API.

This sample allows the user to view the acceleration forces along the X-, Y-, and Z-axes for a 3-axis accelerometer. You can choose one of three scenarios:

  • Accelerometer data events
  • Accelerometer shake events
  • Current accelerometer reading

Acclerometer Data Events

When you choose the Enable button for the Accelerometer data events option, the app begins streaming accelerometer readings in real time.

Accelerometer Shake Events

When you choose the Enable button for the Accelerometer shake events option, the app displays the cumulative number of shake events each time an event occurs. (The app first increments the event count and then renders the most recent value.)

Polling Accelerometer Reading

When you choose the Enable button for the Current accelerometer reading option, the app retrieves the most recent accelerometer reading.

Build the sample

Building the C++ Version of the Sample

To build this sample, open the solution (.sln) file titled AccelerometerCPP.sln from Visual Studio 2013 Preview. Press F7 or go to Build->Build Solution from the top menu after the sample has loaded.

Building the C# Version of the Sample

To build this sample, open the solution (.sln) file titled AccelerometerCS.sln from Visual Studio 2013 Preview. Press F7 or go to Build->Build Solution from the top menu after the sample has loaded.

Building the JavaScript Version of the Sample

To build this sample, open the solution (.sln) file titled Accelerometer.sln from Visual Studio 2013 Preview. Press F7 or go to Build->Build Solution from the top menu after the sample has loaded.

Run the sample

Running the C++ and C# Versions of the Sample

To run this sample after building it, go to the installation folder for this sample with Windows Explorer and run the executable from the <install_root>\bin\Debug folder. Alternatively, press F5 (run with debugging enabled) or Ctrl+F5 (run without debugging enabled) from Visual Studio 2013 Preview. (Or select the corresponding options from the Debug menu.)

Running the JavaScript Version of the Sample

To run this sample after building it, go to the installation folder for this sample with Windows Explorer and open the default.html file. Alternatively, press F5 (run with debugging enabled) or Ctrl+F5 (run without debugging enabled) from Visual Studio 2013 Preview. (Or select the corresponding options from the Debug menu.)
